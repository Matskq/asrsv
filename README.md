# asrsv

### Git

Install git

    pacman -Sy git

### Cloning

Clone the GitHub repository

    git clone https://github.com/Matskq/asrsv.git
    
Move to the cloned folder

    cd ./asrsv/

### Scripts

Make the scripts usable

    chmod +x ./installer.sh
